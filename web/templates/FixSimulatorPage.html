Mathematics

Fix Values

Fix values are determined by the following:

  • The amount of health units (Hf) of the fixing unit.
  • The fix (F) value of the fixing unit.

Formula

p = 0.05 * F

if p < 0, set p to 0

if p > 1, set p to 1

For each health unit (Hf) of the fixing unit, three random numbers (r) between 0 and 1 are generated. Each time r < p, a fix is counted. The number of fixes divided by 3 is the number of health units the injured unit gains from the fix.

Notes

  • Fixing has a wider range of possible outcomes compared to attacks, making it less predictable.
